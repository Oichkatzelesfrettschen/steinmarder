## Main Classifications

- `canonical`
  - active project docs, findings, and source-of-truth tables
- `implementation-source`
  - remote Mesa and related live source trees
- `validation-lane`
  - dEQP, DXVK, validation logs, benchmark harnesses
- `graphics-archive`
  - relevant but non-canonical legacy notes, helper scripts, and audit bundles
- `canonical-reference`
  - PDFs and hardware reference materials
- `agent-state`
  - `.claude` memory/config roots
- `archive-only`
  - unrelated or weakly-related machine-history material
- `review`
  - stragglers that need a deliberate decision
