## Key Integration Points
- Uses src/gallium/drivers/r600/sfn/ for VLIW5 shader compilation
- Uses src/compiler/spirv/ for SPIR-V -> NIR translation
- Uses src/vulkan/runtime/ for Vulkan common infrastructure
- Uses src/vulkan/wsi/ for window system integration
